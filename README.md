# World War 2026: Eternum

> Jogo de estrategia global com globo interativo e campanha infinita.

## Sobre o projeto

Jogo de estrategia em tempo real com conquista global, producao de recursos, fila de acoes com duracao real (5 min a 10 h), treino de tropas, globo interativo em canvas, diplomacia, retaliacoes, fase alienigena apos 100% de dominacao e save local com export/import. Modos visual Ultra Clean e Facil inclusos.

## Funcionalidades principais

- Globo interativo em canvas com rotas animadas
- Fila de acoes com duracao real e cancelamento parcial
- Producao de recursos, tropas e pesquisa
- Diplomacia, espionagem e ataques avancados
- Retaliacoes inimigas e defesa ativa
- Fase alienigena e campanha infinita
- Save automatico + export/import JSON

## Tecnologias utilizadas

- HTML5 Canvas
- CSS
- JavaScript
- localStorage
- Netlify

## Como executar

Abra index.html no navegador.
